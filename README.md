##Este trabalho apresenta uma solução simples e de baixo custo para monitoramento e controle automático do nível de água em um reservatório para aplicações de curto prazo. Visando reduzir desperdícios e promover praticidade e eficiência, foi desenvolvido um sensor resistivo e, de acordo com a medição, exibe-se o valor em um display LCD e controla-se uma bomba de água para regular o volume, utilizando uma Raspberry PI como microcontrolador. Para garantir a precisão, realizou-se a calibração com base em valores conhecidos, o tratamento dos dados na lógica de programação e projetou-se uma estrutura que pudesse ser acoplada ao recipiente para melhorar a estabilidade. O sistema apresentou bons resultados e mostrou-se viável, apesar de apresentar algumas limitações do modelo proposto.
