# Calibrações realizada nos sensores para compreneder como se pode relacionar as tensões com seus respectivos volumes

## O arquivo calibracao_teste_viabilidade foi realizado na fase inicial do desenvolvimento do projeto para a conpreensão do funcionamento do dos sensor e sua viabilidade, ele foi realizado em dois dias diferentes e sem nenhum suporte impresso, preso somente com fita. Além disso o primeiro teste foi realizado de 100 em 100 ml enquanto oo segundo em 200 e 200 ml.

## O arquivo calibracao_3sens foi a calibração realizada com os 3 possiveis sensores  onde se determinou qual sensor produzia o melhor efeito. O sensor utilizando barra de inox foi posicionado de forma a sobrar 4cm no topo, e ele foi posicionado de forma a ficar cada polo oposto de sim, ou seja a maior distância possivel. O sensor feito com a solda e o sensor feito com o cobre foram posicionados também de forma a ficarem cos os seus polos mais distantes possives   

## O arquivo CALIBRACAO_FINAL foi a calibração final utilizando so a solada e estando conectado no balde somente os aparatos que estariam no projeto final, assim não tendo interferencias dos outros aparatos no volume.
